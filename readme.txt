Vostok is for those who don't want attention to be distracted from content. 
Colors and typography have been carefully chosen to achieve maximum legibility 
with minimum eye fatigue. Also, code has been written with extreme care for 
web standards and accessibility.

Vostok has been designed by Javier Cañada and coded by Rubén Lozano. White
version by Mark MacKay.

**How to install**

Upload both vostok and vostok-white folders into your themes directory
(wp-content/themes).
